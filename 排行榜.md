#### <2>: 按总销售排行

###### 接口:/wms/gross_sales

```
eg: 127.0.0.1:8000/wms/gross_sales?st_time=2019-09-17 12:11:32&en_time=2019-12-17 10:45:56
```

请求参数：

| 字段    | 说明     | 是否必填 |
| ------- | -------- | -------- |
| st_time | 开始时间 | 是       |
| en_time | 结束时间 | 是       |

返回示例：

```
{
    "code": "200",
    "msg": "查询成功",
    "data": [
        {
            "equipment_no": 1,
            "total_fee__sum": 377665
        }
    ]
}
```
