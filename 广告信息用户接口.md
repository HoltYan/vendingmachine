## 广告信息用户接口

------

### 一：接口：base/banner/

##### 1：列表 get

------

##### 2：搜素 get

请求参数：

name：广告名称

------

##### 3：添加 post

参数说明：

| 字段名          | 参数意义 | 是否必填 |
| --------------- | -------- | -------- |
| mode            | mode_add | 是       |
| name            | 名字     | 是       |
| link            | 链接     | 是       |
| Advertisong_img | 图片     | 是       |
| remarks         | 备注     | 否       |

------

##### 4: 修改 post

参数说明：

| 字段            | 说明       | 是否必填 |
| --------------- | ---------- | -------- |
| mode            | mode_alter | 是       |
| id              | 广告id     | 是       |
| remarks         | 备注       | 否       |
| name            | 广告名字   | 否       |
| advertising_img | 图片       | 否       |
| link            | 广告链接   | 否       |



------

### 二：接口：banner_info/

删除广告信息

请求方式：post

| 字段      | 说明         | 是否必填 |
| --------- | ------------ | -------- |
| banner_id | 1或[1,2,3,4] | 是       |

```
{
    "msg": "删除广告信息成功",
    "state": 0,
    "code": "200"
}


```

### 三：接口：device_banner/

查看广告绑定设备详情

请求方式：get



| 字段 | 说明   | 是否必填 |
| ---- | ------ | -------- |
| id   | 广告id | 否       |



请求方式：post



mode_add    # 设备添加广告

mode_alter  # 设备解绑广告

mode_edit  # 修改时间和状态  



| 字段       | 说明               | 是否必填 |
| ---------- | ------------------ | -------- |
| device_id  | 设备id             | 是       |
| banner_id  | 广告id             | 是       |
| status     | 0:上线      1:下线 | 否       |
| start_time | 开始时间           | 否       |
| end_time   | 结束时间           | 否       |
