## 一：对账列表

### 接口：/pay/consume_order

<1>: 对账列表页数据

请求方式：get

请求参数

| 参数                  | 说明                                                         | 是否必填 |
| --------------------- | ------------------------------------------------------------ | -------- |
| code                  | 流水号                                                       | 否       |
| total_fee__gte        | 金额范围大于                                                 | 否       |
| total_fee__lte        | 金额范围小于                                                 | 否       |
| reconciliation_status | 对账状态((1, "已对账"), (2, "未对账"), (3, "异常"), (4, "已冲正"), (5, "存疑")) | 否       |
| pay_method            | 支付方式((1, "未支付"),     (2, "微信"),     (3, "支付宝"))  | 否       |
| merchant__name        | 商户名称                                                     | 否       |

## 二：商户结算

### 接口：/pay/account/

<1>: 商户账户管理

请求方式：get

请求参数：

| 参数    | 说明     | 是否必填 |
| ------- | -------- | -------- |
| name    | 商户名称 | 否       |
| account | 账号     | 否       |

返回示例

```
{
    "code": 200,
    "msg": "查询成功",
    "data": {
        "num": 1,
        "data": [
            {
                "is_delete": false,
                "code": "1",
                "method": 1,
                "name": "mall",
                "account": "1",
                "period": 0,
                "remarks": "hello",
                "create_time": "2020-04-01T13:19:17"
            }
        ]
    }
}
```

接口：/pay/settlement/

<1>: 结算记录列表

请求方式：get

请求参数：

| 参数            | 说明                                                         | 是否必填 |
| --------------- | ------------------------------------------------------------ | -------- |
| code            | 编号                                                         | 否       |
| method          | 支付方式((1, "未支付"), (2, "微信"), (3, "支付宝"), (4, "线下转账"),) | 否       |
| settlement_time | 结算时间                                                     | 否       |
| state           | 结算状态(1, "待结算"), (2, "已结算")                         | 否       |
| remark          | 备注                                                         |          |

返回示例

```
{
    "code": 200,
    "msg": "查询成功",
    "data": {
        "num": 1,
        "data": [
            {
                "is_delete": false,
                "code": "10001",
                "name": "mall",
                "num": 10,
                "total": 100,
                "method": 1,
                "start_time": "2020-03-02T13:28:04",
                "end_time": "2020-04-01T13:28:22",
                "person": "auto",
                "settlement_time": "2020-04-01T13:28:33",
                "state": 1,
                "voucher_img": null,
                "remark": null,
                "create_time": "2020-04-01T13:28:51"
            }
        ]
    }
}
```

<2>: 上传结算凭证

请求方式： post

请求参数

| 参数    | 说明     | 是否必填 |
| ------- | -------- | -------- |
| person  | 结算人   | 是       |
| voucher | 结算凭证 | 是       |
| code    | 单号     | 是       |
| remark  | 备注     | 否       |
